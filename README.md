# Block Code Style
A small style for list of code examples

## The Style
[style.styl](https://github.com/JaisonPeres/css-utils/blob/master/style.styl)

Create a style for ```<code>``` tag in a list.
 Ex: 
 ```html
 <pre class="code">
   <code> let counter = 0 </code>
   <code> counter ++ </code>
 </pre>
 ```
