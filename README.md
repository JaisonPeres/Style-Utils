# Style Utils
small resources for html manipulation with css

## Block Code
[block_code.styl](https://github.com/JaisonPeres/css-utils/blob/master/block_code.styl)

Create a style for ```<code>``` tag in a list.
 Ex: 
 ```html
 <pre class="code">
   <code> let counter = 0 </code>
   <code> counter ++ </code>
 </pre>
 ```
